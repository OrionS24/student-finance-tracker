# Student Finance Tracker

## Overview

Student Finance Tracker is a responsive web application designed to help students manage their personal finances. The application allows users to record transactions, track spending habits, monitor budget limits, search financial records, and back up data using JSON files.

This project was built using **HTML**, **CSS**, and **Vanilla JavaScript** without external frameworks.

---

## Features

### Dashboard

* Displays total transactions
* Shows total amount spent
* Displays most frequently used category
* Calculates remaining budget

### Transaction Management

* Add new transactions
* Edit existing transactions
* Delete transactions
* Real-time transaction search

### Input Validation

* Description validation
* Amount validation
* Date validation
* Duplicate word detection

### Settings

* Currency selection (USD, EUR, RWF)
* Monthly budget cap configuration
* Import data from JSON files
* Export data to JSON files

### Accessibility

* Screen reader announcements
* Skip-to-content navigation
* Semantic HTML structure

### Responsive Design

* Mobile-friendly layout
* Tablet optimization
* Desktop optimization

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in a web browser.
4. Start adding and managing transactions.

No installation or additional dependencies are required.

---

## Data Storage

The application stores:

* Transactions
* Selected currency
* Monthly budget cap

using the browser's Local Storage. Data remains available after refreshing the page.
